Bienvenue au fichier d'aide pour Métamorphose!

Comment utiliser Métamorphose:
  • Dans le panneau Ramasseur, cliquer le bouton Parcourir pour sélectionner un chemin d'accès. Il est aussi possible d'entrer ou de coller un chemin puis de cliquer sur le bouton OK. Le contenu de ce chemin sera alors affiché.

  • Par défaut, tout le contenu du chemin est sélectionné quand le chemin est chargé, pour (dé)sélectionner des fichiers ou dossiers il suffit de les cliquer. Toutes sélections utiliseront les mêmes règles pour être renommées. Si des dossiers sont sélectionnés, leurs contenu ne sera pas changé.

  • Choisir les opérations de renommage qui vous conviennent à partir du panneau -l'Essentiel-. Certaines opérations peuvent être raffinés d'avantage dans le panneau Numération et le panneau Date et Heure.

  • Quand les paramètres des opérations sont changés, leurs résultats sont calculés et affichés automatiquement. Si ce comportement n'est pas désirable, dé-cocher la case Prévisualisation automatique, et les changements seront affichés en cliquant sur le bouton Prévisualiser.

  • Dans le panneau Erreurs/Avertissements, les erreurs qui empêchent le renommage de s'effectuer sont affichées en rouge, les avertissements en jaune.
  • Si tout a l'air bien, cliquer sur le bouton Allons-y!. Votre sélection sera alors renommée.

  • Vous avez une possibilité de remettre la sélection à l'état initial une fois l'opération terminée.

La plupart des parties du logiciel montrent un petit panneau d'aide si le pointeur de la souris est laissé au-dessus pendant quelques instants.
Les autres panneaux de cette fenêtre contiennent plus de détails sur toutes les opérations et options disponibles.
Cette fenêtre peut être gardée ouverte pendant que le logiciel est utilisé.
