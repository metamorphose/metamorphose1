Welcome to the Métamorphose help!

How to use Métamorphose:
  • in the Picker tab, click the BROWSE button to select a path. You may also enter in or paste a path and press the OK button. The contents of this path will be displayed.

  • By default, the entire contents of the path are selected whenever a new path is entered, you can deselect files or folders by clicking on them. All selections will use the same renaming rules. Any folders you select will not have their contents changed.

  • Select any number of renaming options from the - Main - tab. Additionally, you may further refine options by using the Numbering and the Date and Time tabs.

  • Items will be previewed automatically as you change settings. If you do not like this behavior, uncheck Auto Preview, and items will be previewed by clicking on Preview.

  • Errors that will prevent the renaming operation from completing succesfully are highlighted in red, warnings are in yellow. The Errors/Warnings tab will display them.

  • If preview looks OK then click Go! button. At this point your selections will be renamed.

  • You have 1 undo level in case of mistakes.

Most parts of the program will display a help tip if you let your mouse cursor linger a bit over it.
Also definitly check out the other tabs of this help window to see options in more detail!
You can keep this help dialog open and still access all other program funtions.
