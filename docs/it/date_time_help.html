Il pannello Data & Ora permette la gestione delle impostazioni della data e l'ora.


Potrai avere le informazioni di data e ora dagli oggetti invece di settarne di propri. TUTTE le formattazioni funzionaranno.
Note: nei filesystems POSIX (Linux, OS X, BSD, etc), non c'é la data di creazione, siccome non esiste in questi sistemi. Invece potrai vedere Modifica Metadata (ctime) che è l'ora dell'ultima modifica (Ma non necessariamente - i.e. cambiare i permessi cambierà i metadata, ma non modifichera l'ora)




Usa il Calendario per specificare la tua data (il default è la data attuale).
