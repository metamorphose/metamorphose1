Aquí está un ejemplo:
Entrada: %a, %d %b %Y %H.%M.%S +0000
Resultado: mar, 06 feb 2007 20.06.29 +0000

NOTA: No se permite utilizar los dos-puntos « : » porque este caracter es utilizado por Métamorphose para definir el inicio de una operación especial.
