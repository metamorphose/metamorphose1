Leame de Métamorphose

Esta es basicamente una copia de la página de ayuda del sitio web de Métamorphose, así que puede visitar este sitio para obtener información más actualizada.
No obstante, como no todos tienen acceso a internet, aquí está:


Instalación desde el archivo setup.exe:
Aquellos usuarios Win32 quienes no deseen lidiar con los fuentes pueden simplemente descargar el archivo de instalación y correrlo. Métamorphose se instalará como una aplicación win32, y será accesible desde el menú de inicio.


Instalación desde las fuentes:
Instalando la versión apropiada de wxPython (algunos sistemas tal vez ya lo tengan instalado):

  • Python 2.4 o 2.5 disponible desde http://www.python.org/
  • wxPython 2.6 disponible desde http://www.wxpython.org/.
    A partir de la version 0.3.0, Métamorphose trabajará SOLO con la versión unicode!
  • Los paquetes de Debian necesarios para wxPython son: libwxgtk2.6, python-wxgtk2.6


Instalación:
Descomprima las fuentes en el directorio deseado. Si usted usa Linux o freeBSD, vaya al directorio y escriba (como root):

$ make all

Como correrlo:
Ejecute 'Metamorphose.py' con el interprete Python dentro de la carpeta 'Metamorphose'.
A partir de la versión 0.3.0, puede especificar un directorio para abrir donde comenzar.
Los espacios están permitidos, por lo que la siguiente linea es perfectamente válida. 
$ python Metamorphose.py /srv/samba/Windows Gusta Los Espacios

Alguna información extra para Linux:
El fichero makefile debería encargarse de todo para usted, es independiente de la arquitectura y de la distribución.
Métamorphose se instalará en /usr/share/metamorphose, lo puede correr con 
$ metamorphose
y la página de ayuda (manpage) con 
$ man metamorphose
Si esta usando un gestor de ventanas compatible con freedesktop.org (como Gnome o KDE), debería haber una entrada en 'Aplicaciones -> Accesorios'.


Como remover Métamorphose:
Windows:
Corra el desinstalador o remuevalo desde 'Agregar o quitar programas' del panel de control.

Fuentes (todos):
Simplemente elimine los archivos descomprimidos durante la instalación.

Fuentes (Linux & FreeBSD): 

$ make remove
Para borrar también todos los archivos de los usuarios : 
$ make remove remusr=1

Corriendo la distribución Compilada:
Windows 95, 98, ME: Estable con Métamorphose v0.2.3a y MENORES. A partir de la versión 0.3.0 hay algunas cosas, como caidas cuando se intenta acceder a los menues superiores.
Windows 2k, XP, 2k3: Estable, soporta themes.

Corriendo desde las fuentes:
Windows 2k, XP, 2k3: Estable, soporta themes.
Linux 2.6 - GNOME: Estable, soporta themes.
Linux 2.6 - KDE: Estable.
Mac OS 10.4: Estable, soporta themes.
FreeBSD 5.5 - Window Maker:Estable en su mayor parte. Eligindo cerrar mientras se cambia el lenguaje puede bloquear la aplicación, pero el lenguaje es cambiado adecuadamente una vez que matas el proceso y lo reinicias. Además, cuando los combos se desplegan los eventos no se disparan correctamente.
FreeBSD 6.2 - Fluxbox: Estable.


Algunas cuestionesa saber:
La aplicación se bloquea cuando recorre un gran número de archivos/carpetas:
Realmente no se bloquea, pero el tiempo que tarda el proceso puede ser muy largo si está cargando muchos archivos y/o directorios. Durante este proceso la aplicación no refresca, dando la apariencia de que se ha bloqueado pero actualmente se está trabajando en esto.
El tiempo que permanece en este estado depende del desempeño de el ordenador, de si el directorio es local o si es un directorio compartido de una red, y por supuesto del número de items a procesar.

Python error - "cannot set locale to 'xx_XX' ":
Aplicable en desktops Linux solamente. Verifique que el idioma sea admitido por el sistema, porque algunas distribuciones (como Ubuntu) no instalan soporte para otros idiomas excepto los especificados durante la instalación.
