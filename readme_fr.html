Instalation avec le fichier executable:
Les utilisateurs de win32 peuvent simplement télécharger le fichier executable et le lancer. Métamorphose sera instalé comme n'importe autre logiciel pour Windows, et sera disponible du menu démarage.


Instalation d'après les sources:
Pré-instaler la version correcte de wxPython (certain systèmes peuvent déjà avoir ceci):

  • Python 2.4 ou 2.5 disponible sur www.python.org
  • wxPython 2.6 disponible sur www.wxpython.org.
    À partir de la version 0.3.0, Métamorphose marchera UNIQUEMENT avec la version unicode!
  • Paquets pour Debian nécéssaires à wxPython: libwxgtk2.6, python-wxgtk2.6

Instalation:
Décomprimer le fichier contenant les sources où vous les voulez. Si vous utilisez Linux ou freeBSD, allez à ce répertoire puis entrez (comme root): 
$ make all

Comment lancer:
Executer 'Metamorphose.py' avec l'interprète Python dans le dossier 'metamorphose'.
À partir de la version 0.3.0, vous pouvez spécifier un dossier à ouvrir.
Les espaces sont permis, donc le suivant est parfaitement acceptable: 
$ python Metamorphose.py /srv/samba/Windows Aime Les Espaces

Un peu plus d'information pour Linux:
Le fichier makefile devrait tout faire pour vous, il est indépendant de l'architecture et de la distribution.
Métamorphose sera instalé dans /usr/share/metamorphose, vous pouvez le lancer avec 
$ metamorphose
et la page guide (manpage) avec 
$ man metamorphose
Si vous utilisez un gestionnaire de fenêtres compatible avec freedesktop.org (comme Gnome ou KDE), il devrait avoir une entrée dans 'Applications -> Accessoires'.


Désinstalation de Métamorphose:
Windows:
Lancer le désinstaleur ou enlever à partir d' 'ajouter/enlever programmes' dans le panneau de contrôle.

Sources (tous):

  • A partir de la version 0.4.0, supprimer les fichiers particuliers à l'utilisateur:
    Windows: C:\Documents and Settings\UTILISATEUR\Application Data\.metamorphose
    Linux: /home/UTILISATEUR/.metamorphose
    Mac: /Library/Application Support/.metamorphose
  • Supprimer la page guide (manpage) si vous l'avez installée.
  • Supprimer les fichiers sources.
Sources (Linux & FreeBSD) : 
$ make remove
Pour aussi supprimer tous les fichiers des utilisateurs : 
$ make remove remusr=1

Systèmes d'exploitation testés:
Execution Binaire:
Windows 95, 98, ME: Stable avec Métamorphose v0.2.3a et INFÉRIEUR. À partir de v0.3.0 il y a quelques problèmes, comme un crash en accedant les menus supérieurs.
Windows 2k, XP, 2k3: Stable, support de thèmes.

Execution interpretée:
Windows 2k, XP, 2k3: Stable, support de thèmes.
Linux 2.6 - GNOME: Stable, support de thèmes.
Linux 2.6 - KDE: Stable.
Mac OS 10.4: Stable, support de thèmes.
FreeBSD 5.5 - Window Maker: Plutôt stable. Choisir de fermer en changeant la langue peut planter l'application, mais la langue est changée correctement une fois que l'application est tuée et redémarrée. Aussi les listes déroulantes peuvent ne pas déclencher un événement correctement.
FreeBSD 6.2 - Fluxbox: Stable.


Quelques problèmes connus:
Le logiciel se gèle en 'cheminant' beacoup de fichiers/dossiers:
Pas vraiment gelé, mais l'attente peut être longue si beaucoup de fichiers/dossiers sont chargés. Pendant ce processus, l'application ne se rafraîchi pas, donnant l'impression d'être gelée.
Le temps dans cet état est dépendant de votre ordinateur, si le chemin est local ou sur réseau, et bien sûr du nombre d'articles.

Python error - "cannot set locale to 'xx_XX' ":
Applicable aux desktops Linux seulement. Verifiez que la langue est supportée par le système, car certaines distributions (comme Ubuntu) n'installent pas de support pour d'autres langues à part celles spécifiées durant l'installation.
