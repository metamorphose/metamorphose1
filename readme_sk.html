Métamorphose Prečítaj si

Poznámka: Toto je kópia nápovedy z web stránky Métamorphose, kde nájdete aktuálne informácie.


Inštalácia zo setup.exe súboru:
Uživatelia Win32 si môžu namiesto zdrojovej distribúcie stiahnuť a spustiť setup súbor. Métamorphose sa nainštaluje rovnako ako akýkoľvek Win32 program, a bude prístupný zo štart menu.


Inštalácia zo zdrojových súborov:
Je potrebné mať nainštalovanú správnu verziu wxPython (niektoré distribúcie ju majú):

  • Python verzie 2.4 alebo 2.5 je dostupný z www.python.org
  • wxPython 2.6 je dostupný zwww.wxpython.org.
    Od verzie .3.0 Métamorphose pracuje iba s Unicode verziou!
  • Debian balíčky potrebné pre wxPython: libwxgtk2.6, python-wxgtk2.6

Inštalácia:
Rozbaľte súbory do vybraného adresára. Pokiaľ ste na platforme Linux alebo freeBSD, choďte do rozbaleného adresára a zadajte (ako root) príkaz:
$ make all

Spustenie:
Spustite 'metamorphose.py' s Python interpreterom z rozbaleneho 'metamorphose' adresára.
Od verzie 0.3.0 môžte zadať adresár, ktorého obsah sa má načítať.
Medzery sú dovolené, takže napríklad toto bude fungovať: 
$ python Metamorphose.py /srv/samba/Windows Likes Spaces

Extra Linux informácia:
makefile sa postará o všetko, je nezávislý na architektúre a distribúcii.
Métamorphose bude nainštalovaný do /usr/share/metamorphose, program spustíte príkazom 
$ metamorphose
manuálovú stránku zobrazíte príkazom 
$ man metamorphose
Ak používate window manager kompatibilný s freedesktop.org (ako Gnome alebo KDE), tak by sa Métamorphose malo objaviť v 'Applications -> Accessories'.


Ako odstrániť Métamorphose:
Windows Inštalátor:
Spustite odinštalovať alebo odstrániť z 'pridať alebo odstrániť programy' z kontrolného panelu.

Zdrojové súbory (všetky):

  • Začínajúc s verziou 0.4.0, všetky uživateľské dáta sa odstránia zmazaním adresárov:
    Windows: C:\Documents and Settings\USERNAME\Application Data\.metamorphose
    Linux: /home/USERNAME/.metamorphose
    Mac: /Library/Application Support/.metamorphose
  • Zmazať manuálovú stránku, ak bola inštalovaná.
  • Zmazať zdrojové súbory.
Zdrojové súbory (Linux & FreeBSD): 
$ make remove
Navyše pre odstránenie všetkých uživateľských dát: 
$ make remove remusr=1

Testované operačné systémy:
Spustené skompilované:
Windows 95, 98, ME: Stabilné od Métamorphose v0.2.3a and nižšej. Od verzie 0.3.0 sa vyskytú problémy, ako pády aplikácie pri prístupe na horné menu.
Windows 2k, XP, 2k3: Stabilné, podporuje témy.

Spustené zo zdrojákov:
Windows 2k, XP, 2k3: Stabilné, podporuje témy.
Linux 2.6 - GNOME: Stabilné, podporuje témy.
Linux 2.6 - KDE: Stabilné.
Mac OSx86: Stabilné, podporuje témy.
FreeBSD 5.5 - Window Maker: Prevažne stabilné. Výber reštartu aplikácie pri zmene jazyka môže zamrznúť aplikáciu. Ale jazyk je správne zmenený potom, čo je aplikácia zabitá a znovu spustená. Taktiež dropdown box-y nie vždy správne spúšťajú udalosti.
FreeBSD 6.2 - Fluxbox: Stabilné.


Známe problémy:
Program zamrzne pri prechádzaní veľkého množstva súborov/adresárov:
Program v skutočnosti nezamrzne, iba trvá dlhšiu dobu než sa načítajú všetky položky. Počas tejto doby sa aplikácia neobnovuje, takže sa javí ako zamrznutá, ale v skutočnosti pracuje. Takto strávený čas závisí na výkone počítača, na tom či daný adresár je lokálny alebo vzdialený, a samozrejme na počte položiek.

Python chyba - "nedá sa nastaviť locale na 'xx_XX' ":
Týka sa to iba Linux desktopov. Uistite sa, že máte nainštalovanú podporu daného jazyka, pretože niektoré distribúcie (ako Ubuntu) defaultne neinštalujú jazykovú podporu iných jazykov, než jazyka použitého pri inštalácii.
